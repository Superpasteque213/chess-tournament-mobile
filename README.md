#Chess Tournament Mobile

##Objectif
- L’objectif de ce projet est la création d’une application android de gestion de tournois d’echecs, elle doit permettre de créer et gérer des tournois voir les anciens en respectant les recommandations officielles de la FIDE et ensuite on pourra rajouter des fonctionnalités

##Fonctionnalités principales
- Créer un tournoi
- Ajout/Suppression de joueurs
- Appariements ronde après ronde 
- Génération des résultats ronde après ronde, possibilités de les exporter

##Contraintes
- Android studio – langage Kotlin
- Interface simple, efficace et ergonomique
- Utiliser le système Suisse et respecter les normes FIDE
